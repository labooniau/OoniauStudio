# Ooniau Studio One-Page Homepage

This is a static one-page homepage for Ooniau Studio and TWUSStock.

## Files

- `index.html`: one-page website.
- `en/index.html`: English one-page website.
- `styles.css`: visual styling for the page.
- `assets/ooniau_studio.png`: studio image used by the page.
- `assets/screenshots/`: five TWUSStock app screenshots for the promotion section.
- `assets/screenshots-en/`: five English TWUSStock app screenshots for the English page.
- `GOOGLE_SITES_COPY.md`: paste-friendly content for rebuilding the page in Google Sites.
- `GOOGLE_SITES_COPY_EN.md`: paste-friendly English content for Google Sites.

## Preview

Open `index.html` in a browser for Traditional Chinese.

Open `en/index.html` in a browser for English.

## Google Sites Use

Google Sites does not always preserve custom HTML/CSS as a full page. For a native Google Sites page, copy the sections from `GOOGLE_SITES_COPY.md`, upload `assets/ooniau_studio.png` as the main image, and upload the five images in `assets/screenshots/` for the app screenshot section.

For the English page, copy `GOOGLE_SITES_COPY_EN.md` and upload the five images in `assets/screenshots-en/`.

Use the published Google Sites URL as your studio homepage and Privacy Policy URL only if the page keeps the privacy policy, data handling, financial disclaimer, and contact sections visible to the public.

## Compliance Notes

The copy avoids:

- Investment return promises.
- Claims that quote data is real-time or guaranteed.
- Implying that TWUSStock is a broker, trading platform, wallet, lender, or asset custodian.
- Saying data never leaves the device, because quote lookup sends ticker/search requests to third-party services.
